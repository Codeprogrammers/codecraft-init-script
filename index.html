Welcome to the GitHub Page for the Codecraft-init-script project.

The latest version is 0.4.3

Changes in this version include:

  • Bugfix for lockfile on Debian/Ubuntu-based systems
  • Bugfix for ramdisk failing to unmount
  • Changes to command-line arguments *
  • Code cleanup

* the arguments "worldbackup" and "rdbackup" have been replaced with "backup worlds" and "backup ramdisk" respectively. See the following:

$ service minecraft backup {worlds|ramdisk}
